# ProjectPilot AI

ProjectPilot AI is an AI agent that converts a rough project idea into a complete AI/ML project blueprint.

## Features

- Generates project title
- Creates problem statement
- Suggests dataset requirements
- Recommends ML/AI approach
- Provides project workflow
- Suggests evaluation metrics
- Includes basic safety guardrails
- Maintains session history using Streamlit

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## How to Run

```bash
python -m streamlit run app.py
